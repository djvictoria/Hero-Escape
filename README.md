# Hero-Escape
A 2D platformer about a hero surviving the dangerous lands outside the castle.

# Controls
**WASD:** Movement<br/>
**Left Click:** Attack (Click repeatedly for a 2-hit combo)<br/>
**Shift:** Sprint<br/>
**Space:** Jump
